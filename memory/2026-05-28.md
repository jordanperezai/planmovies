### Key Decisions

- **Page DNA:** Hero visual → social proof → action → context. Codified in DESIGN.md.
- **Auth is opt-in post-RSVP:** Name + going + party size = RSVP. Phone = optional update preference.
- **Side panel rejected by 3/3 personas** — Tia Rosa: "I'd press the home button." Marco: "Side panels on mobile are annoying." Sofia: "Don't hide the reason behind the ask."
- **Two tabs → one page (pending build):** 3/3 personas preferred merge. "The Movie" tab becomes anchor link to timeline section.
- **Timeline + Category view toggle** — timeline for narrative weight (chronological), category view for pattern recognition (six presidents in a row). Mockup has both.
- **Separate tabs killed the argument** — the current Movie page's three-act structure (movie → reality break → evidence) is broken by tab separation. Merging restores it.
