## Key Decisions
- Cloudflare over Vercel (Browser Rendering ecosystem for future automation)
- Supabase over Convex/Firebase (CDN client, already connected via MCP)
- Single HTML file, no framework (ships fast, works everywhere)
- Regal Secaucus locked as venue (no IMAX, all recliners, 14 screens)
- "Seven billion" not "eight billion" (plot detail, one billion aren't human)
- Level 2 automation: monitor + alert, manual purchase in batches of ~10
- Stripe for payment collection, not Venmo (real checkout, Apple Pay support)
- Ticket price set at $22 (evening ~$16-17 + convenience fee + buffer)
