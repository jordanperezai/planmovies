# memory/

Session logs and knowledge graph. One file per day: `YYYY-MM-DD.md`. Multiple sessions in the same day are sections within the file.

**Rules:** Write-only. Grepped for context, never fully read. Written by /wrap-up at session end.

## Structure

- `YYYY-MM-DD.md` -- session logs (one per day, `## Session N` sections within)
- `topics/` -- knowledge graph nodes (extracted from MEMORY.md when 3+ entries cluster)
