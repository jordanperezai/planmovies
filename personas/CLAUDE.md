# Personas

Testing panels for validating decisions. Each persona reacts independently to a stimulus. No debate between them. The synthesis finds patterns across reactions.

Different from rangers/ (adversarial debate where voices argue with each other).

## Active Panels

None yet. Panels created when a decision needs validation against real-world archetypes.

## Structure

```
personas/
  [panel-name]/
    roster.md          Who the personas are
    memory.md          Validated principles + dead directions (optional)
    sessions/          Session transcripts
```
